# my-writing-samples/ — 你本人的英文筆法素材（跨論文共用）

把**你自己**過去寫過、已發表或滿意的英文論文／段落，或個人寫作指引放在這裡。`style-calibration` skill 會讀取這些素材來學習你偏好的語氣與筆法，避免把稿件改得不像你的風格。這是跨所有論文共用的一層。

## 版控說明（重要）

**本資料夾的內容是私人素材，已被 [`.gitignore`](../../.gitignore) 忽略、不進版控**——只有本 `README.md` 會進版控。你放進來的檔案只留在本機。

⚠️ **請勿把第三方版權素材**（例如出版教材、他人文章）放到任何會公開的位置；這類內容留本機參考即可，不要 commit、不要 push。

## 對照：預設風格基準

跨論文的**預設風格基準**（資管 IS 論文通用預設：語域、時態、拼寫變體等）在 [`../style-defaults.md`](../style-defaults.md)，那份是公開框架的一部分、會進版控；本夾則是**你個人**的筆法素材。兩者一起餵給 `style-calibration` 產生每篇專屬的 `style-guide.md`。
